Scenario-Based NPV Analysis Model

An advanced Python and Excel-based financial modelling tool designed to evaluate investment opportunities through Net Present Value (NPV) analysis under varying economic conditions. The model enables users to input custom cash flows and discount rates while assessing project performance across bull, bear, and base-case scenarios.

Overview

This project was developed to simulate real-world investment analysis by combining Excel-based financial modelling with Python-driven analytical capabilities. The system helps users evaluate how changing market conditions and discount rates influence project profitability, investment feasibility, and risk exposure.

Capital budgeting

Scenario forecasting
